# 🕰️ The Lost Legacy: Chronicles of Arkana (1920s Retro Edition)

Sebuah permainan pengembaraan retro berasaskan pelayar web (HTML5/JS) dengan estetika tahun 1920-an. Terokai dunia misteri, selesaikan teka-teki, dan kumpulkan serpihan Arkana Crystal.

##  Ciri-ciri Permainan
- **Estetika Retro 1920-an:** Efek filem lama (film grain), warna sepia, dan gaya visual vintage.
- **Watak Realistik:** Watak utama dilukis secara terperinci (jaket kulit, goggles, turtleneck).
- **Mekanik Lengkap:** Panjat, Tunduk, Lompat, dan Lawan.
- **5 Tahap Pengembaraan:** Dari Hutan Purba hingga ke Teras Arkana.
- **Sistem Simpanan:** Autosave menggunakan LocalStorage pelayar.

## 🕹️ Cara Bermain (Controls)
| Kekunci | Tindakan |
|---------|----------|
| `W` / `↑` | Lompat / Panjat |
| `S` / `↓` | Tunduk (Crouch) |
| `A` / `←` | Bergerak Kiri |
| `D` / `→` | Bergerak Kanan |
| `SPACE` | Lompat |
| `F` | Serang / Lawan |
| `E` | Interaksi |
| `TAB` | Inventory |
| `ESC` | Pause Menu |
